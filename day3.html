Styling Methods

Inline

Inline styling allows the user to add a style attribute to an element and give a specific snippet of code a particular style
One of the drawbacks is that multiple style properties clutters up the code and can slow browser load times

Internal

A secondary method is known as internal styling
This method allows the user to include style properties in the head of the html code
                
<HTML>
 <head>
     <title>Styling Methods</title>
     <style>
       body {
         color: green;
         /*This gives the font color in the body a color of green*/
       }

       h1 {
         text-align: center;
         background-color:blue;
         width: 300px;
       }
    </style>
 </head>
 <body>

     <h1>
       My main header has a width of 300px, background color of blue and the text is aligned to the center
     </h1>

 </body>
</HTML>

External

Internal styling gives the user more room to write in styles, but can clutter the head of the html code
External styling is the most common practice
External styling requires the creation of an external css stylesheet and then linking the file inside the html head
                
<HTML>
 <head>
     <title>Styling Methods</title>
     <link rel="stylesheet" href="css/styles.css">
     <!--
       link tags require the stylesheet reference in the rel attribute and the path of the css document in the href attribute
       -->
 </head>
 <body>

     <h1>
       My main header
     </h1>

     <p class="p_class">
       My P tag has a class attribute that will be referenced back to the styles.css document
     </p>

     <div class="box-1">
     </div>

     <div class="box-2">
     </div>

     <div class="box-3">
     </div>

 </body>
</HTML>
the styles.css document would look like:
              
    /*The Beginning of your styles.css document*/

    body{
      /* Adding style to the body */
      color: blue;
    }

    h1{
      text-align: center;
      color: green;
      /* Targeting the h1 font color overrides the body's color*/
    }

    .p_class{
      width: 300px;
      background-color: red;
    }

Styling Concepts

Positioning

Web development requires attention to positioning and space
It is important for the developer to be in control of the elements on the page
Thus, it is good practice to consider positioning when creating elements on a page
The styling properties, width, height, float and clear are core to controlling the positioning of elements
              
  /*Your styles.css document */
  /*Each class below is attached to div's on your html document*/
  .box-1 {
    width: 300px;
    height: 500px;
    float: left;
    background-color: green;
  }

  .box-2 {
    width: 300px;
    height: 500px;
    float: left;
    background-color: red;
  }

  .box-3 {
    width: 900px;
    height: 200px;
    float: left;
    clear: both;
    background-color: orange;
  }

  /*The above classes create three div's:
    the first two boxes are of equal size but have different bg-colors
    the third box is of a different size and sits below the other two boxes
  */

Spacing

Spacial awareness is another core concept when styling
Spacing requires the developer to think about how the elements are going to fit
The margin, border and padding style properties are core to handling space on your page
              
  /*Your styles.css document */
  /*Each class below is attached to div's on your html document*/
  .box-1 {
    width: 300px;
    height: 500px;
    float: left;
    background-color: green;

    /*controlling space*/
    margin: 0px 10px 0px 0px;
    padding: 10px;
  }

  .box-2 {
    width: 300px;
    height: 500px;
    float: left;
    background-color: red;

    /*controlling space*/
    margin: 0px 10px 0px 10px;
    border: 1px solid black;
  }

  .box-3 {
    width: 900px;
    height: 200px;
    float: left;
    clear: both;
    background-color: orange;

    /*controlling space*/
    margin: 20px 0px 10px 0px;
    padding: 15px;
    border: 1px solid yellow;
  }

  /*
    - The code above has been changed to include spacing properties, margin/padding/border
    - the first box has a margin to its right of 10px and padding inside of 10px all the way around
    - the second box has a margin to the left of 10px and a black solid border of 1px
    - the third box has a margin on top of 20px and below of 10px, padding inside of 15 px and a 1px solid green border
  */
