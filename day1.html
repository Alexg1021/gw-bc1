What is Hyper Text Markup Language(HTML)?

The standard markup language used to create websites
Programming language that powers the web
The current version is HTML5
HTML allows images and videos to be embedded and can be used to create interactive forms
The language consists of elements/tags enclosed in angle brackets

Tools of the Trade

Building Blocks of a Webpage

     
 <HTML>
    <head>
        <title>My page</title>
    </head>
    <body>
        <span class="someclass" id="someid">Content!</span>
    </body>
 </HTML>

Heading Tags

Heading tags are used as titles or to highlight important text throughout websites
Heading tags are ordered sequentially from h1 - h6, with h1 being the largest and h6 being the smallest
Heading Tags
               
                 <HTML>
                    <head>
                        <title>My page</title>
                    </head>
                    <body>
                       <h1>Header 1</h1>
                       <h2>Header 2</h2>
                       <h3>Header 3</h3>
                       <h4>Header 4</h4>
                       <h5>Header 5</h5>
                       <h6>Header 6</h6>
                    </body>
                  </HTML>
