File Structure

File structure implies the order and organization in which you create your various web based files. It is important to maintain an organized and structured file system so that you may refer to any particular file with efficiency.
An appropriate file structure example:

                              |__img

Nesting

Nesting refers to the concept of code within code
Think of the Russian Dolls as an example, where the larger outer doll houses many smaller dolls inside
The outermost element is referred to as the parent and the sub-elements are referred to as children
An example includes
                 
                   <HTML>
                      <head>
                          <title>Nesting Concepts</title>
                      </head>
                      <body>

                        <!-->Basic Nesting Principles<-->
                        <div class="outer-box" id="level-1">
                            <div class="sub-box" id="level-2">
                                <div class="sub-box" id="level-3"> This is the inner most box and in the third level of the group of code</div>
                            </div>
                        </div>

                        <!-->Another Example: Nesting with elements on the same level<-->
                        <div class="outer-box" id="level-1">
                          <div class="sub-box" id="level-2">
                              <h1 class="sub-box" id="level-3">Header on the third level of the nested code </h1>
                              <p class="sub-box" id="level-3">paragraph text on the same level as the header</p>
                          </div>

                          <div class="sub-box" id="level-2">
                              <h2 class="sub-header" id="level-3">Header on the third level of the nested code</h2>
                              <ul class="sub-list" id="level-3">
                                  <li class="sub-list" id="level-4">The overall list is on the third level of this group of code</li>
                                  <li class="sub-list" id="level-4">But each list item is nested inside the ul an thus on the fourth level</li>
                              </ul>
                          </div>
                        </div>


                      </body>
                    </HTML>

Styling

To style an element use the style attribute in an element
This allows the user greater control of the code, but using the style attribute has its drawbacks
                 
                   <HTML>
                      <head>
                          <title>Nesting Concepts</title>
                      </head>

                        <!-->Using the Style Attribute to Style html<-->

                                <div class="sub-box" id="level-3" style="color: red;" > This is the inner most box and in the third level of the group of code</div>
